Legacy codes are 8 digits.

For the materials, it's always starts with 1 (1st digit).

The next 2 digits (2nd, 3rd) represent the main category of this material, the next 2 digits (4th, 5th) represent subcategory of this material, the last 3 digits represent the item itself.

For example, these are materials from the main category 01 التبريد:

10101001
10101002
10101003
10102001
10102002
10102003
10102004

These are materials from the main category 02 الكهرباء:

10201001
10202002
10202001
10203001
10203002
10203003

These are materials from the main category 03 الغاز:

10301001
10302002
10302003
10302004
10302005

(Of course, each one of theme includes subcategories)

Special-case stock files (stock/special-case/)

Files placed under data/materials/raw-materials/stock/special-case/ are excluded from the normal
extract-raw-materials.ts stock scan (which only reads the flat stock/ directory). Each file there
must be handled by a dedicated extract script.

Subcategory 12/02 ("زجاج راكد") has no legacy codes in codes.xls. Its materials are extracted
directly from stock/special-case/راكد الزجاج.xls via:

  npm run extract:stagnant-glass

Output: data/materials/raw-materials/results/clean-stagnant-glass.csv
Seeded with legacyCode = null (no official 8-digit legacy code exists for these items).
